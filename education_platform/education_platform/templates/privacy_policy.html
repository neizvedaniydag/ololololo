Политика конфиденциальности

Дата последнего обновления: 27 октября 2025 г.

1. Общие положения

Настоящая Политика конфиденциальности разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки и защиты персональных данных пользователей образовательной платформы (далее — Платформа).

ВНИМАНИЕ: Данная платформа является учебным проектом, созданным в образовательных целях. Все данные обрабатываются исключительно для демонстрации функционала образовательной системы.

2. Оператор персональных данных

Оператором персональных данных является владелец данной платформы. Для связи: [укажите email]

3. Какие персональные данные собираются

При регистрации на платформе собираются следующие данные:

  • Имя пользователя — для идентификации в системе
  • Электронная почта (email) — для входа в систему и связи
  • Пароль — хранится в зашифрованном виде (bcrypt-хеш)

4. Цели обработки персональных данных

Персональные данные обрабатываются исключительно для следующих целей:

  • Регистрация и авторизация пользователей на платформе
  • Предоставление доступа к образовательным материалам
  • Сохранение результатов прохождения тестов и физических упражнений
  • Генерация персонализированных тестов с помощью ИИ (GigaChat API)
  • Ведение статистики успеваемости пользователя

5. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • Согласия субъекта персональных данных (пользователя), предоставленного при регистрации
  • Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных»

6. Порядок и условия обработки

Способы обработки: сбор, запись, систематизация, накопление, хранение, уточнение, извлечение, использование, обезличивание, блокирование, удаление, уничтожение.

Хранение данных: Персональные данные хранятся в базе данных на сервере. База данных защищена от несанкционированного доступа. Пароли хранятся только в зашифрованном виде (bcrypt).

Локализация данных: В соответствии с требованиями законодательства РФ, все персональные данные пользователей размещаются на серверах, находящихся на территории Российской Федерации.

7. Передача данных третьим лицам

Платформа использует API сервиса GigaChat (Сбер) для генерации образовательного контента. При генерации тестов в GigaChat передаётся только информация о предмете и теме — персональные данные пользователя (имя, email) не передаются.

Данные не передаются третьим лицам в коммерческих целях и не используются для рекламы.

8. Использование cookies

Платформа использует cookies только для обеспечения работы системы авторизации (Flask-Login). Эти файлы необходимы для сохранения сессии пользователя и не содержат личной информации, кроме идентификатора сессии.

9. Сроки обработки персональных данных

Персональные данные обрабатываются в течение всего срока существования учетной записи пользователя. После удаления аккаунта данные удаляются в течение 30 дней.

10. Права субъекта персональных данных

Вы имеете право:

  • Получать информацию о хранящихся персональных данных
  • Требовать уточнения или удаления неточных данных
  • Отозвать согласие на обработку персональных данных
  • Удалить свою учетную запись вместе со всеми данными

Для реализации своих прав обращайтесь по адресу: [укажите email]

11. Безопасность данных

Оператор принимает следующие меры для защиты данных:

  • Пароли хранятся в зашифрованном виде с использованием алгоритма bcrypt
  • Доступ к базе данных ограничен
  • Используется протокол HTTPS для защиты передачи данных (при размещении на продакшене)
  • Регулярно проверяется актуальность систем безопасности

12. Изменение политики конфиденциальности

Оператор имеет право вносить изменения в настоящую Политику. Новая редакция вступает в силу с момента её размещения на сайте. Пользователи будут уведомлены о существенных изменениях.
